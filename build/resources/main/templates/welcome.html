Rules of The Game:

  • Hangman is a guessing game. You get a mystery word and have to guess it, letter by letter!
  • The number of dashes on the screen represent each letter in the word.

  • Once a letter is guessed correctly, all spots containing that letter will be revealed.
  • Each guess adds to your (unlimited!) guess total.
  • If you guess a correct letter but it has already been revealed, this will add to your guess total.
  • If you guess an incorrect letter, that will add to your incorrect guess count.
  • Your maximum number of guesses will be listed once you start a game.

  • If you guess the correct word, the game is over and you win!
  • If you run out of guesses, you lose.

  • To rejoin a previously created game, add its Game ID to your URl
  • For example: .../game/2

  • For more information, visit: Hangman (game) Wikipedia Page



Most importantly: Have Fun!
